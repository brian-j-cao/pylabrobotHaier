# Quick Start Guide

Get up and running with the Haier Liquid Handling System in just a few minutes.

## Prerequisites

Before you begin, ensure you have the following:

- [ ] System requirements met (see [Installation Guide](installation.md))
- [ ] Access credentials
- [ ] Network connectivity

## Step 1: Installation

1. Download the latest release from [releases page]
2. Run the installer with administrator privileges
3. Follow the setup wizard

```bash
# Example installation command
./install-liquid-handling.exe
```

## Step 2: Initial Configuration

1. Launch the application
2. Run the configuration wizard
3. Set up your first liquid handling protocol

```yaml
# Example configuration
system:
  name: "Haier Liquid Handler"
  version: "1.0.0"
  mode: "production"
```

## Step 3: First Protocol

Create your first liquid handling protocol:

1. Open the Protocol Designer
2. Add liquid handling steps
3. Validate and save the protocol
4. Run a test execution
